## Getting Started

To run / develop locally:

1. **Clone the repository**  
   ```bash
   git clone https://github.com/alvrian/alvrian.github.io.git
   cd alvrian.github.io
   ```

2. **Install dependencies** (if it uses a build step)  
   ```bash
   npm install
   ```

3. **Run locally**  
   ```bash
   npm run dev
   ```

4. **Add or update content**  
   - Projects / Certifications: update the relevant JSON / markdown / config files  
   - Images: place assets in the appropriate folder  
   - Update navigation as needed  

5. **Build / Preview**  
   ```bash
   npm run build
   ```
