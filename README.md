## About

This portfolio website serves as a living showcase of my:

- Academic achievements  
- Technical projects  
- Professional experience  
- Certifications & publications  

It is designed to be easy to update and maintain as I grow in my career.

## Features

- Responsive UI (mobile & desktop friendly)  
- Interactive sections for projects, blog / publications (if applicable)  
- Clean, modern design, emphasizing readability and visuals  
- Organization that allows easy addition of new items (projects, certifications, etc.)  
- Hosting using GitHub Pages via `alvrian.github.io`
